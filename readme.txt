=== ThreeWP Login Tracker ===
Tags: network, wpms, wpmu, login, logins, password, tracker, list, last login 
Requires at least: 3.0
Tested up to: 3.0
Stable tag: trunk

Tracks all logins and login attempt on a blog or network.

== Description ==

Maintains a list of logins for all users. Logs successful and unsuccessfull logins for each user.

Viewing / deleting of login data can be restricted to specific user roles.

See screenshots for more info.

Available in
* English
* Swedish

== Installation ==

1. Unzip and copy the zip contents (including directory) into the `/wp-content/plugins/` directory
1. Activate the plugin sitewide through the 'Plugins' menu in WordPress.

== Screenshots ==

1. Login Tracker menu item and login statistics column in the user list
1. Each user has a list of logins and login attempts
1. The list of latest logins with user IP and user agent info 
1. Overview of all the users and how many logins or login attempts they have
1. Settings
1. Uninstall
